# 定时器属性

![1562144978567](assets/1562144978567.png)

* Name : 对象名
* Enabled : True / False ( 定时开关 )
* GenerateMember : 是否生成成员变量
* Interval : 事件频率 ( 单位 : ms )
* Modifiers : 可见级别
